Developing your environment.

Lets start by creating the file structure for the site. type the following in the desired directory: django-admin startproject [name] I'll be calling this project "thiswebsite" so I'll type: (django-admin startproject thiswebsite) Make sure that you step inside the thiswebsite directory: cd thiswebsite When you list the cwd you should see(manage.py, thiswebsite)
